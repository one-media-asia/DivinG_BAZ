# Scuba Admin - Diving Administration System

A Flask-based web application for managing diving operations, divers, equipment, and dive sites.

## Setup

1. Clone the repository
2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```

3. Install dependencies (once requirements.txt is created)
4. Run the setup script to initialize the database:
   ```bash
   python setup.py
   ```

5. Start the application:
   ```bash
   python app.py
   ```

## Features

- User authentication and role-based access control
- Diver management
- Dive site management
- Equipment tracking
- Dive log administration

## Technologies

- Flask
- SQLAlchemy
- SQLite
- Flask-Login
